# **App Name**: CareerCompass AI

## Core Features:

- User Profile Builder: A structured, multi-step form to capture user's education, skills, experience, and career goals.
- Skill Scoring Engine: Convert raw user inputs into normalized skill scores (0-5) for DSA, CS Fundamentals, Projects, and Experience.
- Company Knowledge Base: Maintain structured profiles of companies, including required skill thresholds, weightage per skill, interview patterns, and hard reject rules.
- Candidate-Company Fit Calculator: Calculate fit percentage based on weighted scoring and reject rules to provide Apply / Prepare / Avoid decisions.
- Personalized Roadmap Generator: Generate a time-bound, adaptive preparation roadmap, prioritizing high-ROI topics and excluding low-ROI ones.
- "What NOT To Do" Guidance: Provide explicit negative advice on skills, companies, and preparation paths to avoid, with clear reasoning.
- AI-Powered Explanation Tool: Explain the results of the matching engine in human-friendly language, formatting roadmaps and insights.

## Style Guidelines:

- Primary color: Deep Blue (#1E3A8A) to convey trust and professionalism.
- Background color: Light Gray (#F9FAFB) to provide a clean and modern look.
- Accent color: Teal (#2DD4BF) for highlighting key actions and insights.
- Body font: 'Inter' sans-serif for a clean, modern, neutral style suitable for large blocks of text.
- Headline font: 'Space Grotesk' sans-serif for a computerized, techy feel.
- Use clean, professional icons from a consistent set (e.g., Material Design Icons) to represent skills, topics, and company attributes.
- Dashboard-driven layout with clear sections for user profile, skill scores, company recommendations, roadmap, and progress tracking.
- Subtle transitions and animations to provide feedback on user actions and data updates.
